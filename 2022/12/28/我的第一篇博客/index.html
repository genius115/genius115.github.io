我的第一篇博客

1、创建博客文章

  1. $ hexo new “我的第一篇博客”
  2. 用软件typora打开直接编写文章
  3. 清理hexo clean
  4. 构建静态文件hexo g
  5. 上传至仓库(上传报错多位网络原因，多上传几次即可)hexo d

2、创建草稿文件

  1. 新建草稿文件hexo new draft b
  2. 预览草稿文件hexo server –draft
  3. 发布草稿hexo publish b

3、GitHub上建站访问

  1. 新建github仓库

    仓库名称限制了为你的：用户名+.github.io

  2. 安装hexo上传插件

  3. 修改hexo配置文件指定仓库路径

    可在文件夹中直接打开文件，也可通过vim直接编辑

  4. 推送站点到github

  5. 尝试访问

    输入你的仓库名称,即可访问成功。

    https://用户名+.github.io/
